  • Speech Recognition Videos, Graphics Videos, Hardware and Art Videos
    Many videos are available through my Vimeo account documenting projects such as Mathy, a completely speech-operated iPad App built to help solve a series of math problems by asking it as you would a person. There are two videos of two separate speech recognition engine examples tuned to different language models, one using the Declaration of Independence, and one using Romeo and Juliet, with each being read the same set of lines, showing how what one has heard in the past affects what one hears in the present.
    There are graphics demos such as the kaledoscopic Spiraling Shape, which uses camera input to change the input to the shader, and Dragonfly Wings, another GPU shader based fractal animation. I also have a video of an interactive Newton Fractal Explorer. There are also many videos documenting my Masters project, Forces of Nature, which uses a grid of electromagnets to manipulate a vessel of ferrofluid under computer control through an iOS based UI used by a human performer. A much longer demo including a view of the user interface on iOS, showing which electromagnets are active at which times. There are also two public performance videos. Experiments with a different type of ferrofluid vessel have also proven interesting.
  • Interactive SVG Data Visualization Charts
    I designed and programmed a generation system to visualize a large, multi-dimensional dataset and a set of filters that categorize the data. The SVG chart example includes interactive zoom and horizontal panning capabilities, and shows multiple filters and the dataset in an intuitive way.

  • Un-named Javascript Window Mangaer
    A prototype of an AJAX window manager I worked on for the codehearted site, it is built as a layer on top of dynAPI. My code here supports click-dragging of windows, close-box functionality, and clicking to bring a window to front (layering). Note that this code was written in 2002, the current version of Safari and Chrome show some cosmetic issues while dragging, and the close boxes no longer function in 2018 browsers. But overall, for a 16 year old web tech demo, it functions pretty well.
