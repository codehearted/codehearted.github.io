The Systems Area is located just north of the Main Stacks, near the east-west center of the second floor. Systems includes offices and laboratories for the library's technical staff, as well as the server room.

Hardware and software testing and development is performed within systems, as is the location of the main server room. There are five (5) standard Staff Workstations, and one (1) Staff Workstation with attached Scanner. There is a single (1) Staff Printer as well. Many servers (n) and some other special equipment will be located in the server room, but the exact number must be calculated following system usage estimates being obtained. For more information, see the Server Room section of the Miscellaneous Hardware Notes page.

To the northwest is Public Services Area, and to the southeast is the Main Stacks Area. Administration is to the northeast.
