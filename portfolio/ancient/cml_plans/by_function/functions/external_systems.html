External Systems Interface

Cliolink 2001 will connect with several other systems to accomplish it's tasks.

For the exchange of circulation data, it will interface with both the Checkpoint Security System and the Bestseller OPAC server. Access to these critical library databases is expected to be limited to read-only.

The system will also recieve environmental information from the SmartBuilding system, and integrate it into the Cliolink environment as is deemed appropriate. For example, the graphical layout of the library that a user sees while browsing through library areas in the tour may indicate the ambient room temperature on the area map.

Cliolink will connect to several external content databases on the behalf of users, caching results system-wide where possible to lower external network bandwith usage.
