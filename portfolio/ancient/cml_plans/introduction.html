Cerritos Millennium Library

Introduction


This Project Description Document is being released to provide a model for and provoke further discussion on the Cerritos Millennium Library's goals, progress. It is not a comprehensive description of the systems, functions, and machines that are going to be involved in the library's operation, although it is hoped to eventually evolve into a complete high-level description of at least the digital systems planned to be in operation on Opening Day.

There are several sections in this release that have not been completed, this is to save time, as the missing information would be redundant to information already contained in other sections and must be verified with library staff. You are part of this verification process. Several places where there will be more extensive cross-referencing (for example, the left frame in any area page) are not yet linked. However, the navigational system present in this version should be adequate to find one's way around.

Known Issues:

  • More detail is to be added to the Function and Machine views.
  • Left frames in area pages have no links to machines, other areas, or functions.
  • Notes for areas (right frame) have no links or descriptions of functions for the area.
  • Function pages do not yet reference all areas and machines that are involved in that function.
  • Servers are not yet represented in the system, other than a note on the misc. hardware notes page that the servers will be located in the server room on the second floor, in the Systems area (machine types and quantity not yet available)
  • Area maps not clickable to go to adjacent areas, or machines
  • Function icons are unnattractive.
  • This page, the contents page, and the area index don't look good under netscape.

If you have issues or concerns about information presented in this document that are not listed above, please email paul@tmbg.org (Paul Jacobs), or talk to him in person.
