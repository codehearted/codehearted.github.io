Cliolink Printing and Scanning Hardware

Printing is done primarily through the Cliolink Print Servers, which can redirect requests to any network-based printer in the system. Some machines have attached printers and/or scanners, which are not available for printing to from other machines. Machines with attached printers may, however, print to other networked printers.
